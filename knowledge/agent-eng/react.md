# ReAct

**Statement:** Interleave reasoning traces with actions, grounding each thought in real observations.

**Detail / real params:** ReAct alternates reasoning steps and actions so each thought is grounded in an actual observation from the environment rather than a prediction. This is the algorithmic antidote to hallucination: observe the real output instead of guessing what it will be.

**Source:** https://arxiv.org/abs/2210.03629
