# Four ACI design rules (verbatim)

**Statement:** SWE-agent's four concrete, copyable rules for a good agent surface.

**Detail / verbatim official rules:**
1. **Guardrail:** a linter blocks edits when "code isn't syntactically correct."
2. **File viewer** shows "just 100 lines in each turn."
3. **Search** must "succinctly list the matches" — more context is "too confusing for the model."
4. **Empty output:** on a command that produces nothing, return "Your command ran successfully and did not produce any output."

**Source:** https://swe-agent.com/latest/background/aci/
