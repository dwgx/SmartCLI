# Agent-Computer Interface (ACI) thesis

**Statement:** A purpose-built Agent-Computer Interface — not just the model — determines agent success.

**Detail / real params:** SWE-agent introduces the ACI: simple, LM-centric commands and feedback formats designed for the model rather than for humans. Reported results: pass@1 of 12.5% on SWE-bench and 87.7% on HumanEvalFix, versus non-interactive LMs. The thesis: interface design drives outcomes.

**Source:** https://arxiv.org/abs/2405.15793
