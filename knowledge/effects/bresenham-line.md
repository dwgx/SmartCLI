# Bresenham's line algorithm

An integer-only line rasterizer that steps along the major axis choosing pixels with an error accumulator; used to draw wireframe edges into a character grid.

## Formula / params
- Integer error-term line drawing between two endpoints.
- Used to draw the 12 edges of the rotating cube.
