# Palette: SynthWave '84

Neon retro theme (robb0wen SynthWave '84 for VS Code); saturated cyans/magentas over dark plum backgrounds.

- Exact hex: `#241b2f #2a2139 #03edf9 #36f9f6 #72f1b8 #ff7edb #f97e72 #fe4450 #fede5d #848bbd`

## Usage
Roles: `#241b2f`/`#2a2139` = background plums; `#03edf9`/`#36f9f6` = neon cyans (highlights, active); `#72f1b8` = mint (strings/success); `#ff7edb` = hot magenta (the signature glow, keywords); `#f97e72`/`#fe4450` = coral/red (warnings/errors); `#fede5d` = yellow (accents); `#848bbd` = muted lavender (comments/inactive). For an animated neon glow, cycle the two cyans and the magenta as gradient stops via [[color-interpolation]]; pair with [[hsv-cycling-lolcat]] for a moving sheen. High saturation over dark plum is the whole aesthetic — keep backgrounds dark.
