问题所在

你的 AI 在终端里是盲的。

把一个全屏程序接上管道，你什么也得不到，甚至直接卡死。 改用 SmartCLI 来驱动，模型就能读到语义化的屏幕快照，用方向键 移动，并逐步核验每一步。

开始使用

解压，放进去，直接就能用。

内核从 PyPI 安装；技能原地即可运行。一个 拖放进来的 drive-tui 文件夹甚至自带一份内核。

# 共享内核，来自 PyPI pip install smartcli-toolkit # 然后，在 Python 里 from smartcli_core import PtySession # 或者检查一个拖放进来的技能文件夹是否接好了 python skills/drive-tui/scripts/tui.py doctor
